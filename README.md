# Weather Application

This project is a React-based weather application that utilizes the OpenWeatherMap API to display current and forecasted weather information.

## Features

- Display weather for the user's current location
- Search for weather in any city
- Show current weather, hourly forecast, and 5-day forecast
- Allow switching between different days in the forecast
- Toggle between Celsius and Fahrenheit
- Responsive design

## Tech Stack

- React
- Redux
- TypeScript
- React Router DOM
- Styled Components

## Getting Started

1. Clone the repository
2. Install dependencies:

```bash

  npm install

```

3. Rename the `.env.example` file to `.env`
4. Start the development server:

```bash
   npm run start

```

## Project Structure

- `src/components/`: React components
- `src/pages/`: Page components
- `src/store/`: State management (Redux)
- `src/styles/`: Styled Components files
- `src/utils/`: Utility functions and helpers
- `src/types/`: TypeScript type definitions
- `src/hooks/`: custom hooks
